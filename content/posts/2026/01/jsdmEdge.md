## 前言
最近对于网站的字体加速，vercel虽然好用但是用了代理就不能添加`header`的配置项，所以看向了EdgeOne。本身自己有免费版但是放到海外的腾讯云上，那么就来看如何配置出属于自己的镜像吧。

## 添加网站
这边得话可以自己去看下如何配置网站中的套餐，这里不在赘述。只需配置以下这几部分就行了，其余默认
``` md
加速域名: `填入项`.你的根域名
源站配置: `IP/域名`: `fastly.jsdelivr.net`
回源配置: HTTPS
```

## 站点加速
以下为缓存引擎的具体内容
### IF块 1 (针对特定域名和Host重写)
| 配置项 | 条件/操作 | 参数1 | 参数2/运算符 | 值/目标值 |
| :--- | :--- | :--- | :--- | :--- |
| **匹配条件 1** | 匹配类型 | HOST | 等于 | `jsdmirror.jyluo.top` |
| **匹配条件 2** | 操作 | Host Header 重写 | 自定义 | `fasily.jsdeliv.in.net` |
| **触发操作** | 操作 | 强制缓存 TTL | 设置为 | `2 天` (并开启“强制缓存”) |

### IF块 2 (针对特定路径、后缀和正则匹配)
| 配置项 | 条件/操作 | 参数1 | 参数2/运算符 | 值 |
| :--- | :--- | :--- | :--- | :--- |
| **匹配条件 1** | 匹配类型 | URL path | 等于 | `/npm*`, `/gh*`, `/combine*`, `/china*`, `/cdn-cgi*` || |
| **匹配条件 2** | 匹配类型 | 文件后缀 | 等于 | `m3u8`, `ts`, `m3u`, `mpd` |
| **匹配条件 3** | 匹配类型 | URL path | 正则匹配 | `/(npm\|gh\|china)(\.\.)?(jpg\|jpeg\|gif\|ico\|png\|bmp\|webp\|psd\|gif\|tiff\|svg\|avif\|exif\|hei\|heic\|png\|cr2\|nef\|orf\|raw\|png\|jpg\|png\|css\|png\|mpg\|psd\|` |
| **触发操作** | 操作 | 自定义 Cache Key | 模式：查询字符串 | 选择“全部替换” |

### IF块 3 (针对全部请求的全局设置)
| 配置项 | 条件/操作 | 类型/头部名称 | 设置/值 |
| :--- | :--- | :--- | :--- |
| **匹配条件** | 匹配类型 | 全部 (站点任意请求) |  |
| **触发操作 1** | 操作 | Brotli 压缩 Gzip 压缩 | 开启 (根据Content-Type智能识别) |
| **触发操作 2** | 操作 | 修改 HTTP 响应头 | 类型: **设置** <br> 头部名称: **Cache-Control** <br> 值: `public, max-age=31536000, s-maxage=31536000, stale-while-revalidate=31536000` |
